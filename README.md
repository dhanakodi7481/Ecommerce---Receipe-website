# 📖 About

**Recipe E-Commerce Website** is a full-stack Django application where users can discover delicious recipes, browse food categories, add products to their cart, and complete purchases through a responsive online shopping experience.

This project demonstrates Django's authentication system, product management, shopping cart functionality, and database integration.

# 🚀 Installation

## Clone Repository

```bash
git clone https://github.com/dhanakodi7481/Ecommerce---Receipe-website.git
```

## Navigate to Project

```bash
cd Ecommerce---Receipe-website
```

## Create Virtual Environment

```bash
python -m venv venv
```

### Windows

```bash
venv\Scripts\activate
```

### Linux / macOS

```bash
source venv/bin/activate
```

---

## Install Dependencies

```bash
pip install -r requirements.txt
```

---

## Apply Database Migrations

```bash
python manage.py migrate
```

---

## Run Development Server

```bash
python manage.py runserver
```

Visit the application at:

```
http://127.0.0.1:8000/
```
